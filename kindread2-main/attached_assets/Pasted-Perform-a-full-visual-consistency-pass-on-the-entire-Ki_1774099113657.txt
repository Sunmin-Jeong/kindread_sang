DESIGN TOKENS — apply everywhere, no exceptions:

Colors:
  --green:        #2A7A52
  --green-bg:     #E8F3ED
  --green-mid:    #C0DECE
  --text:         #0A0A0A
  --text-2:       #444444
  --text-3:       #888888
  --border:       #EBEBEB
  --surface:      #F7F7F7
  --white:        #FFFFFF
  --red:          #EF4444
  --amber:        #F59E0B
  --purple:       #7C3AED
  --purple-bg:    #F3F0FF

Typography:
  Display / headers:  Playfair Display (no italic)
  All UI text:        Inter
  Quotes only:        Playfair Display, green color
  NO italic anywhere else

Spacing unit: 20px horizontal padding on all screens
Border radius:
  Cards:    12px
  Buttons:  10px
  Pills:    20px
  Avatars:  50%
  Covers:   5-6px

Borders: 1px solid #EBEBEB (light), never heavier
Shadows: 0 2px 8px rgba(0,0,0,0.08) for covers/cards only

GLOBAL COMPONENTS — standardize these:

Top bar (every screen except Feed):
  Height: 52px top padding + content
  Left: ← Back button (Inter 14px 500, color #0A0A0A)
    — EVERY screen that is not a root tab MUST have this
  Center: page title (Inter 15px 600)
  Right: action button if needed (Inter 13px 600, color --green)
  Border-bottom: 1px solid #EBEBEB
  Background: white, sticky

Avatar:
  Sizes: 28 / 34 / 44 / 58 / 72px depending on context
  Border-radius: 50%
  Streak dot (if streak >= 3): 🔥 in 13x13px white circle,
    bottom-right, box-shadow: 0 0 0 1px #eee

Tag/Badge pills:
  Font: Inter 11px 500
  Padding: 3px 9px
  Border-radius: 20px
  Variants: green, grey, purple, orange, blue
  (defined above)

Buttons:
  Primary: bg #0A0A0A, white text, border-radius 10px, padding 14px, full-width
  Secondary: bg white, border 1.5px #EBEBEB, same sizing
  Green CTA: bg --green, white text (for Join, Create etc.)
  All: Inter 14px 600, font-family Inter

Section labels:
  Inter 10px 700, uppercase, letter-spacing 0.1em, color #888

CONSISTENCY RULES:

1. Every non-root screen needs ← Back in top-left
2. Loading states: use skeleton shimmer, never raw spinner on blank white
3. Empty states: friendly message + illustration/emoji, never just plain text
4. Korean UI labels: translate all to English
   (user-generated content stays as-is)
5. All section labels: Inter 10px 700 uppercase
6. No ALL CAPS headings like "RECENTLY TAGGED" or "MY CLUBS"
   — use section-label style instead
7. Star ratings: always amber #F59E0B filled, #E5E5EA empty
8. Progress bars: always 3px height, --green fill, #EBEBEB background,
   border-radius 99px
9. Book covers in lists: always with subtle shadow
   (0 2px 8px rgba(0,0,0,0.08)), border-radius 5px
10. All async operations: try/catch/finally,
    never leave UI in permanent loading state
