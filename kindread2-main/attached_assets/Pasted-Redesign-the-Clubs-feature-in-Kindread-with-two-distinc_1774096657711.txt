Redesign the Clubs feature in Kindread with two distinct club types: 
Session (one-time) and Club (ongoing). All text must be in English.

---

PART 1: DATA MODEL

Update the clubs table in Supabase:
  - type: 'session' | 'club' (required)
  - format: 'offline' | 'online' | 'read_along' (required)
  - book_id: uuid FK to books (required for sessions, optional for clubs)
  - meet_date: timestamptz (for sessions and club meetups)
  - location: text nullable (for offline)
  - link: text nullable (for online)
  - deadline: timestamptz nullable (for read_along sessions)
  - session_number: int nullable (for ongoing clubs, tracks which session)

Add a club_sessions table for ongoing clubs:
  - id, club_id, session_number, book_id, format
  - meet_date, location, link, deadline
  - status: 'upcoming' | 'active' | 'completed'

Add a club_votes table for next book voting:
  - id, club_id, book_id, proposed_by (user_id), created_at

Add a club_vote_responses table:
  - id, vote_id, user_id, created_at
  (one row per user per vote = one vote per user per book candidate)

PART 5: CREATE FLOW

Entry point: + button on Clubs list header

STEP 0 — Type select screen:
  Title: "What kind of club?" (Playfair Display, 26px)
  Subtitle: "Choose how you want to read together."

  Two option cards (full-width, border, borderRadius 16px):

  SESSION card:
    - Icon: 📚, Name: "Session", Tag: purple "One-time"
    - Description: "Built around a single book. Meet once, then it's done."
    - Example tags: 📍 Offline meetup · 🖥 Online discussion · 📖 Read-along
    - Selecting → goes to Session create flow

  CLUB card:
    - Icon: 🌿, Name: "Club", Tag: green "Ongoing"  
    - Description: "A recurring group. Pick new books each session, 
      or explore movies, essays, anything."
    - Example tags: 📚 Book club · 🎬 Film club · ✍️ Writing group
    - Selecting → goes to Club create flow

SESSION CREATE (3 steps, shown as progress dots):

  Step 1 — Format:
    "How will you meet?"
    3-column grid: 📍 Offline / 🖥 Online / 📖 Read-along
    Each option: icon + label + subtitle, border, borderRadius 12px
    Selected state: green border + light green background

  Step 2 — Book:
    "Which book?"
    Search input → shows results from books table
    Each result: cover + title + author, green checkmark when selected

  Step 3 — Details:
    "When and where?"
    - Date input (always shown)
    - If Offline: Location text input
    - If Online: Link text input
    - If Read-along: Deadline date input
    - Session name input (optional, placeholder: "e.g. Steppenwolf read-along")
    
    Submit: "Create Session ✓" → creates club with type='session',
    navigates to session detail

CLUB CREATE:
  Same format selection + name + description + cover image upload
  + Require approval toggle
  + Welcome message template (optional)
  → Creates club with type='club', session_number starts at 1
